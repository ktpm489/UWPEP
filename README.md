# Visual Explorer

UWP练手项目，支持分栏视图的UWP资源管理器。

## 功能:

1. 基本的文件操作功能，如地址栏，打开文件夹，打开文件，重命名，删除文件
2. 能在平铺视图和分栏视图两种模式中切换
3. 右侧有可随时打开的侧栏用于快速预览文件信息
4. 支持多窗口使用
5. 支持文件和文件夹的拖拽操作
6. 支持选中文件后空格预览常见文本文件，图片文件和视频文件
